## Run Locally

Clone the project

```bash
  git clone https://github.com/SaharshPatel24/DMGPRoject.git
```

Go to the project directory

```bash
  cd DMGPRoject
```

Start the server

```bash
  npm run start
```

Open link in browser

```bash
http://localhost:3000/
```
